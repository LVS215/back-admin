# Blog Platform API

Полнофункциональная платформа для ведения блога с REST API.

## Особенности

- 🔐 JWT аутентификация
- 📝 CRUD для статей, комментариев, категорий
- 📧 Email нотификации
- 🔍 Поиск и фильтрация
- 📊 Аналитика просмотров
- ⚡ Кэширование с Redis
- 🔄 Фоновые задачи с Celery
- 🐳 Docker контейнеризация
- 📈 Мониторинг и логирование
- 🔒 Защита от DDoS и брутфорса
