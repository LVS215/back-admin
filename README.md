# Blog API

Полнофункциональное API для блога на Django с использованием Django Ninja и Django REST Framework.

## Особенности

- 🔐 **JWT аутентификация** с refresh токенами
- 📝 **CRUD операции** для статей, комментариев, категорий
- 🔍 **Поиск и фильтрация** статей
- 👍 **Лайки/дизлайки** для статей и комментариев
- 📌 **Закладки** для статей
- 📊 **Статистика** и аналитика
- 📱 **Пагинация** и сортировка
- 🏷️ **Теги и категории** с иерархией
- 🚀 **Docker контейнеризация**
- 📈 **CI/CD** пайплайн
- 📝 **Полная документация API**
- 🧪 **Обширное тестирование**

## Технологии

- Python 3.10+
- Django 4.2
- Django Ninja
- Django REST Framework
- PostgreSQL 15
- Redis
- Celery
- Docker & Docker Compose
- Nginx
- Gunicorn
